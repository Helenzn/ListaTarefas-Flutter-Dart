# Lista de Tarefas em Flutter-Dart

**Bem-vindo(a) ao meu repositório ListaTarefas-Flutter-Dart!** 

- Aqui você encontrará uma lista de tarefas
- Utilizado Flutter e Dart
  
## Instalação para utilização

- Copie o código do ```main.dart```
- Cole na IDE online: [link](https://dartpad.dev/)
- Clique em Run para visualizar 

## Sobre o projeto

- Modelagem de Dados da Classe Tarefa
- Gerenciamento de Estado StatefulWidget
- Entrada de Dados
- Estrutura da Interface: Column, Row, Expanded, ListView.builder (Lista Dinâmica)
- Interações CRUD: criar, ver, editar e remover tarefa
